# Contributing to RootOps

Thanks for your interest in contributing! We welcome improvements to code, docs, examples, and integrations.

## Getting Started
- Fork the repo and create a feature branch
- Set up a virtual environment and install dependencies
- Run `uvicorn src.main:app --reload` to develop locally
- Write tests when adding new functionality

## Pull Requests
- Keep changes focused and small
- Update documentation and examples where relevant
- Ensure CI passes
- Reference related issues in your PR description

## Coding Guidelines
- Python 3.11+
- Async I/O for network and DB operations
- Type hints and Pydantic models for APIs
- Log meaningfully; avoid sensitive data in logs

## Issues
- Use the issue templates for bugs and features
- Provide reproduction steps and expected behavior
